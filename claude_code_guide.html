2️⃣ Claude-in-Chrome（Chrome拡張機能）の役割

🟡 Chrome拡張機能とは？
Claude CodeがChromeブラウザを「手として使うための橋」です。
これがあることで、Claude CodeがChromeを自動操作できます。
🌐
ページを開く・閉じる
👆
ボタンをクリック
⌨️
文字を自動入力
👀
画面の内容を読む
📊
データを取得する
→ Supabaseのセットアップやデータ確認に活躍しました！

7️⃣ 人に教えるときの 3つのポイント

💡 「Claude AIは口だけ、Claude Codeは手も動く」
ふつうのAIはアドバイスするだけ。Claude Codeは実際にファイルを作ったり修正したりできる。

💡 「GitHubは保管＋公開、Supabaseはデータ共有」
GitHubはHTMLファイルを置いて誰でも見られるURLを作る場所。Supabaseは入力したデータをMac・iPhoneで共有する場所。

💡 「Chrome拡張は、ClaudeがChromeを操作するための橋」
Claude CodeがブラウザでSupabaseやGitHubの設定を自動でやるために必要な拡張機能。
